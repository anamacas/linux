1. O que é Linux, quem desenvolveu e suas distribuições?

O que é Linux?

Linux é um sistema operacional gratuito, rápido e seguro, usado em computadores, servidores e celulares Android.

Quem criou?

O Linux foi criado por Linus Torvalds em 1991.

Principais distribuições:

  • Ubuntu
  • Debian
  • Fedora
  • Linux Mint
  • Arch Linux

2. Histórico de Comandos

O que é o histórico no terminal?

É a lista de comandos anteriormente utilizados pelo usuário.

Como visualizar?

Use o comando: history

Como reutilizar?

  • ↑ seta para cima — comandos anteriores
  • ↓ seta para baixo — comandos mais recentes

3. TAB Completion

O que é autocompletar?

Ajuda a completar comandos ou nomes de arquivos automaticamente.

Como usar?

Pressione TAB ao começar a digitar um comando.

4. Comando man

Para que serve?

Abre o manual explicando em detalhes o uso de cada comando.

Como acessar?

man nome-do-comando

Man pages

São as páginas do manual organizadas por seções.

5. Comando shutdown

Para que serve?

Desligar ou reiniciar o sistema.

Exemplos:

  • shutdown now — desliga imediatamente
  • shutdown -r — reinicia o sistema
  • shutdown +5 — desliga em 5 minutos

6. Comandos Essenciais do Linux

  • pwd — mostra o diretório atual
  • ls — lista arquivos e diretórios
  • Curingas: * e ?
  • cd — navega pelos diretórios
  • mkdir — cria um diretório
  • rmdir — remove diretório vazio
  • touch — cria arquivos
  • cat — exibe conteúdo
  • cp — copia arquivos
  • mv — move ou renomeia
  • rm — remove arquivos
